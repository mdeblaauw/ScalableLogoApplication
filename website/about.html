The motivation of this application

To help people deal with the vast amount of information they face during attendance of a conference, we want to create an application which can automatically store information from an organisation by taking a picture of its logo. Our goal is to create a scalable logo classifier that is widely applicable for conferences. To do this, we created a meta-learning classifier that can learn different sets of logo classes with only five training samples per class. This means that every logo class in this application is prepared with only five training samples.

Infrastructure of this application

This is a serverless AI-driven application constructed with AWS services. This means that no server is running in the backend while there are no requests. However, when there is a request, the required computation needed is automaticly reserved. We did this by deploying a deep meta-learning Pytorch model using primarily AWS Lambda functions.
